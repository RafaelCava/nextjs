# Primeiro sistema com Next.js

## iniciar projeto
- Para iniciar o projeto clone ele para sua maquina ou faça um fork

```bash
git clone <linkdesserepositorio>
```
- Depois instale a dependencias

```bash
npm install 

#or

yarn
```
- Depois é so inserir o comando
```bash
npm run dev

#or

yarn dev
```
